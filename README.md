# NexusNV Web

Simple placeholder website for [NexusNV](https://github.com/nexusnv), owned by
[Azahari Zaman](https://github.com/azaharizaman).

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Scripts

```bash
npm run lint
npm run build
```
